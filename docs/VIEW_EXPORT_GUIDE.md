# 导出文件查看指南

## 概述

导出笔记后，您可以通过导出成功提示中的"查看"按钮快速访问导出的文件。根据不同的设备和权限情况，系统会提供不同的查看选项。

## 查看选项

### 1. 打开文件

直接使用系统默认应用打开导出的 JSON 文件。通常会用文本编辑器或代码编辑器打开。

### 2. 打开文件夹

打开文件所在的文件夹，方便您管理文件或将其复制到其他位置。

- **系统下载目录**：`/storage/emulated/0/Download/`
- **应用下载目录**：`/storage/emulated/0/Android/data/com.ze_memo/files/Download/`

### 3. 查看帮助

如果文件保存在应用专用文件夹，点击"查看帮助"可以了解如何访问这些文件夹。

### 4. 推荐文件管理器

获取推荐的文件管理器应用列表，这些应用能够更好地访问 Android 设备的各个文件夹。

## 不同平台的表现

### Android

#### 已授予完整存储权限
- 文件保存在系统下载目录
- 可以直接打开文件或文件夹
- 支持使用任何文件管理器访问

#### 只有基本存储权限
- 文件保存在应用外部存储目录
- 需要支持访问 Android/data 的文件管理器
- 推荐安装以下应用：
  - Files by Google（Google 官方）
  - Solid Explorer
  - X-plore File Manager
  - Total Commander

#### 没有存储权限
- 文件保存在应用内部存储
- 只能通过应用内部的导出功能访问
- 可以分享到其他应用

### iOS
- 文件保存在应用专用的 Documents/Downloads 文件夹
- 可以通过文件 App 访问
- 支持 AirDrop 分享

### Windows
- 文件保存在用户下载目录：`C:\Users\[用户名]\Downloads\`
- 使用文件资源管理器访问
- 可以直接打开或分享

### macOS
- 文件保存在用户下载目录：`/Users/[用户名]/Downloads/`
- 使用访达访问
- 支持 AirDrop 分享

### Linux
- 文件保存在用户下载目录：`/home/[用户名]/Downloads/`
- 使用文件管理器访问
- 可以通过命令行查看

## 常见问题

### Q: 为什么我找不到导出的文件？

**A:** 这可能是因为：
1. 文件保存在应用专用文件夹
2. 您的文件管理器没有显示隐藏文件或系统文件
3. 需要授予应用完整的存储权限

**解决方案：**
- 打开"查看帮助"获取详细指导
- 安装推荐的文件管理器应用
- 在应用设置中授予"访问所有文件"权限

### Q: 如何将文件移到其他位置？

**A:**
1. 点击"查看" > "打开文件夹"
2. 使用文件管理器复制或移动文件
3. 或者在导出时使用分享功能

### Q: 可以更改默认的导出位置吗？

**A:** 当前版本会根据权限自动选择最佳位置。未来版本将支持自定义导出位置。

### Q: 为什么我的文件管理器无法打开文件夹？

**A:** 某些文件管理器无法访问 Android/data 文件夹。请尝试：
- 安装推荐的文件管理器
- 授予 root 权限（高级用户）
- 使用电脑连接手机访问

## 推荐的文件管理器

### Android

#### Files by Google
- **特点**：Google 官方，免费，支持云存储集成
- **优势**：界面简洁，支持分类浏览
- **下载**：Google Play 商店

#### Solid Explorer
- **特点**：功能强大，支持双窗口
- **优势**：支持 FTP、云存储、加密
- **注意**：部分功能需要付费

#### X-plore File Manager
- **特点**：双窗口设计，经典界面
- **优势**：支持网络存储、压缩文件
- **注意**：广告较多，可付费去除

### Windows

#### Windows 文件资源管理器
- 系统自带，功能完善
- 支持标签页（Windows 11）
- 可安装第三方扩展增强功能

### macOS

#### 访达（Finder）
- 系统自带，深度集成
- 支持标签和智能文件夹
- 可自定义工具栏

## 高级技巧

### 1. 快速访问下载目录
- Android：在主屏幕添加下载目录快捷方式
- Windows：Win + R 输入 `%USERPROFILE%\Downloads`
- macOS：Command + Shift + D 访问下载文件夹

### 2. 批量管理导出文件
- 使用搜索功能查找所有 ze_memo_backup 文件
- 按日期排序组织文件
- 定期备份重要文件到云存储

### 3. 导出文件格式说明
导出的 JSON 文件包含以下信息：
- `exportDate`：导出时间
- `version`：导出格式版本
- `noteFiles`：所有笔记文件数据
- 每个笔记包含分类、内容、时间戳等信息

您可以使用任何 JSON 查看器或文本编辑器查看文件内容。
